# Simple Pixel Art Paint online 
https://dipperok.github.io/pixelartpaint/  
  
ru:
Простой веб-инструмент для создания пиеснль-артов.  
На данный момента имеются возможности и инструменты:  
* Рисовать по холсту кликом или зажатием мыши
* Возможность изменить размер холста (8x8, 16x16, 24x24, 32,32)
* Выбрать цвет из предложенных или через color picker
* Пипетка
* Ластик
* Очистить холст
* Залить определённым цветом фон (не влияет на рисунок)
* Залить все пиксели на холсте
* Скачать картинку
